# encoding-api-using-sentence-transformers
- `sentence_transformers`を使ってテキストのベクトル値を計算するサービスを構築する

## Setup
```sh
# after git clone
cd encoding-api-using-sentence-transformers
poetry install
```

## Usage
```sh
poetry shell
python app.py
```
