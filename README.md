# semantic-trans-search
- [sentence-transformers](https://www.sbert.net/index.html)を使ってベクトル値計算・ベクトル検索を提供するAPIサービスを構築する

## Setup
```sh
# after git clone
cd semantic-trans-search
poetry install
```

## Usage
```sh
poetry shell
python3 ./app.py
```
